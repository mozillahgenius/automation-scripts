# automation-methodology / 業務自動化の方法

## 概要
Google Apps ScriptとAIを活用した業務自動化の方法論ガイド。実現可能性調査から要件定義、コーディング、保守運用までの全工程をカバーするドキュメント。社内業務の自動化プロジェクトを進める際のリファレンスとして活用できる。

## 技術構成
| 項目 | 詳細 |
|------|------|
| 種別 | ドキュメント（方法論ガイド） |
| 対象技術 | Google Apps Script, AI活用 |
| 工程 | 実現可能性調査 → 要件定義 → コーディング → 保守運用 |
| 対象読者 | 業務自動化担当者、情報システム部門 |

## ファイル構成
- `業務自動化の方法` - 業務自動化の方法論を体系的にまとめたドキュメント

## 内容構成
1. **実現可能性調査** - 自動化対象業務の選定と技術的実現可能性の評価
2. **要件定義** - 自動化要件の整理、入出力の定義、関係者との合意形成
3. **コーディング** - GASによる実装、AIツールの活用方法
4. **保守運用** - 運用開始後の監視、改善、トラブル対応

## 使い方
- 新しい業務自動化プロジェクトを開始する際に、工程ごとのチェックリストとして参照
- チームメンバーへの自動化手法の共有・教育資料として活用
