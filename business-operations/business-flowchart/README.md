# business-flowchart / 業務フローチャート図作成

## 概要
Google スプレッドシート上にプログラムで業務フローチャートを描画するGoogle Apps Scriptツール。セルの書式設定、罫線、セル結合を活用して視覚的なフローチャートを自動生成する。ガバナンスフローチャートの作成にも対応。

## 技術構成
| 項目 | 詳細 |
|------|------|
| Platform | Google Apps Script |
| 言語 | JavaScript (GAS) |
| APIs | Google Sheets（SpreadsheetApp） |
| 機能 | セル書式設定、罫線描画、セル結合によるフローチャート生成 |
| トリガー | 手動実行 |

## ファイル構成
- `業務フローチャート図作成.js` - 汎用的な業務フローチャート生成処理
- `governance_flowchart.gs` - ガバナンス業務フローに特化したフローチャート生成

## セットアップ
1. Google Apps Scriptプロジェクトを新規作成（またはスプレッドシートにバインド）
2. フローチャートを描画するスプレッドシートを用意
3. 対象ファイルのコードをエディタに貼り付け
4. スプレッドシートIDやシート名をスクリプト内で設定

## 使い方
- `業務フローチャート図作成.js` の主関数を実行すると、指定シートにフローチャートが自動描画される
- `governance_flowchart.gs` はガバナンス業務フロー専用のレイアウトで描画
- フロー定義（ステップ名、分岐条件等）はスクリプト内で定義・変更可能
