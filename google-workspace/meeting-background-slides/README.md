# meeting-background-slides / 会議背景スライド自動生成

## 概要

会議用の背景スライドを従業員ごとに自動生成するツール。スプレッドシートの従業員データとスライドテンプレートを基に、個人別のスライドを一括作成し Google Drive に保存する。

## 技術構成

| 項目 | 詳細 |
|------|------|
| Platform | Google Apps Script |
| APIs | Google Slides API, Google Drive API, Google Sheets API |
| Triggers | 手動実行 |

## ファイル構成

- `Code.gs` - メイン処理（テンプレート複製、プレースホルダ置換、スライド生成）

## セットアップ

1. Google Apps Script プロジェクトを作成し、`Code.gs` をコピー
2. スライドテンプレートを Google Slides で作成（プレースホルダを含む）
3. 従業員データをスプレッドシートに準備
4. テンプレートスライド ID、スプレッドシート ID、出力先フォルダ ID をスクリプトに設定

## 使い方

- スプレッドシートに従業員情報（名前、部署等）を入力
- スクリプトエディタからメイン関数を実行
- テンプレートが従業員ごとに複製され、プレースホルダが個人情報に置換される
- 生成されたスライドは指定フォルダに保存される
