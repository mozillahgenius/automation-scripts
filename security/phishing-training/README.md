# phishing-training / 標的型メール訓練

## 概要
組織向けの標的型メール訓練を実施するためのGoogle Apps Scriptツール。訓練メールのカスタマイズ、送信対象の管理、開封・クリック追跡を行い、セキュリティ意識向上を支援する。Perplexity APIを活用した訓練メール文面の自動生成機能も搭載。

## 技術構成
| 項目 | 詳細 |
|------|------|
| Platform | Google Apps Script |
| 言語 | JavaScript (GAS) |
| APIs | Gmail, Google Sheets, Perplexity API |
| 機能 | 訓練メール送信、開封追跡、結果集計 |
| トリガー | 手動実行 / スケジュール実行 |

## ファイル構成
- `Code.gs` - メイン処理（訓練メール送信・追跡）
- `PhishingTraining_Complete.gs` - 完成版（全機能統合）
- `PhishingTraining_Advanced.gs` - 高度版（追加機能搭載）
- `Setup.gs` - 初期設定・環境構築用スクリプト
- `appsscript.json` - GASプロジェクト設定
- `標的型メール訓練仕様書` - 訓練の仕様書

## セットアップ
1. Google Apps Scriptプロジェクトを新規作成
2. `Setup.gs` を実行し、初期設定を完了
3. 送信対象者リストをGoogle スプレッドシートに準備
4. Perplexity APIキーをスクリプトプロパティに設定（オプション）
5. 訓練メールのテンプレートをカスタマイズ
6. テスト送信で動作確認後、本番実行

## 使い方
- **訓練準備**: スプレッドシートに対象者（氏名、メールアドレス、部署）を登録
- **メール作成**: テンプレートをカスタマイズまたはPerplexity APIで自動生成
- **訓練実行**: メイン関数を実行し、対象者へ訓練メールを送信
- **結果確認**: スプレッドシートで開封率・クリック率を確認
- **レポート**: 部署別・個人別の結果レポートを生成
