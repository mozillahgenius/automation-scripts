# stock-screener / 株式銘柄チェック

## 概要
日本株の銘柄スクリーニングと定期レポートを自動生成するGoogle Apps Scriptツール。財務指標やテクニカル指標に基づく銘柄フィルタリングを行い、条件に合致した銘柄をメールで定期通知する。

## 技術構成
| 項目 | 詳細 |
|------|------|
| Platform | Google Apps Script |
| 言語 | JavaScript (GAS) |
| APIs | Google Sheets, Gmail, 金融データAPI |
| 対象市場 | 日本株式市場 |
| 機能 | 銘柄スクリーニング、定期レポート、メール通知 |
| トリガー | 時間主導型（定期実行） |

## ファイル構成
- `Code.gs` - メイン処理（スクリーニングロジック、レポート生成）
- `Code_investment_20250828.gs` - 投資判断ロジックの更新版（2025/08/28版）
- `GAS仕様書_株式銘柄チェック.ini` - スクリーニング条件・仕様の定義

## セットアップ
1. Google Apps Scriptプロジェクトを新規作成
2. `Code.gs` のコードをエディタに貼り付け
3. 監視対象銘柄リストをスプレッドシートに登録
4. スクリーニング条件（PER、PBR、配当利回り等）をスクリプト内で設定
5. 通知先メールアドレスを設定
6. 時間主導型トリガーを設定（営業日の市場終了後推奨）

## 使い方
- **スクリーニング実行**: メイン関数を実行すると、設定条件に合致する銘柄を抽出
- **定期レポート**: トリガーにより定期的にスクリーニング結果をメールで通知
- **条件カスタマイズ**: `GAS仕様書_株式銘柄チェック.ini` を参考に、スクリーニング条件を調整
- **バージョン管理**: `Code_investment_20250828.gs` で投資判断ロジックの改良版を管理
