# crypto-portfolio-manager / 暗号資産ポートフォリオ管理

## 概要
CoinGecko APIを活用した暗号資産ポートフォリオ総合管理ツール。ポートフォリオ追跡、アラート設定、取引履歴、積立管理、リバランス提案、価格履歴、ブレイクアウト検知（タートル流）、心理管理、OCO注文、エッジ分析など多機能を搭載。

## 技術構成
| 項目 | 詳細 |
|------|------|
| Platform | Google Apps Script |
| 言語 | JavaScript (GAS) |
| APIs | CoinGecko API |
| データ管理 | Google Sheets（10シート構成） |
| 通知 | Gmail |
| トリガー | 時間主導型 |

## ファイル構成
- `Code.gs` - メイン処理（ポートフォリオ管理、アラート、積立、リバランス、ブレイクアウト検知、心理管理、OCO注文、エッジ分析）
- `appsscript.json` - GASプロジェクト設定

## セットアップ
1. Google Apps Scriptプロジェクトを新規作成
2. `Code.gs` をエディタに貼り付け
3. CoinGecko APIキーをスクリプトプロパティに設定
4. 通知先メールアドレスを設定シートに入力
5. 時間主導型トリガーを設定

## 使い方
- **ポートフォリオ**: 保有銘柄・数量を管理し、現在価値を自動計算
- **アラート**: 価格変動の閾値を設定し、メール通知
- **積立管理**: 定期購入プランの管理・実行
- **リバランス**: 目標配分との乖離を検出し、調整提案
- **ブレイクアウト検知**: タートル流（20日/60日）の価格ブレイクアウトを検出
- **心理管理**: トレード心理状態のトラッキング
- **OCO注文**: 利確・損切の同時注文管理
- **エッジ分析**: トレード戦略の優位性分析
