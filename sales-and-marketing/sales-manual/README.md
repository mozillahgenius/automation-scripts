# sales-manual / 営業マニュアル

## 概要
営業プロセスのDay 0〜Day 10+をカバーする業務マニュアル。Salesforce営業支援システムの設計仕様書、内部統制対応の業務記述書を含む包括的な営業ドキュメント。

## 技術構成
| 項目 | 詳細 |
|------|------|
| 形式 | ドキュメント（Markdown） |
| 連携ツール | Salesforce, Google Drive |
| 対象 | 営業プロセス全般 |

## ファイル構成
- `営業マニュアル` - 営業マニュアル本体（Day 0〜Day 10+の営業プロセス）
- `営業_業務記述書_内部統制対応.md` - 内部統制対応の業務記述書
- `営業支援システム_詳細設計仕様書.md` - 営業支援システムの詳細設計仕様
- `Salesforce営業支援システム_詳細設計仕様書.md` - Salesforce版の詳細設計仕様
- `営業マニュアル_Salesforce_Before-After拡張.md` - Salesforce導入前後の比較・拡張ガイド

## セットアップ
1. 各ドキュメントをチーム共有のGoogle Driveフォルダに配置
2. Salesforceの設定は `Salesforce営業支援システム_詳細設計仕様書.md` を参照
3. 新メンバーには `営業マニュアル` から順に読み進めることを推奨

## 使い方
- **新人研修**: `営業マニュアル` をDay 0から順に実施
- **システム設計**: Salesforce導入・カスタマイズ時に仕様書を参照
- **内部統制**: 監査対応時に業務記述書を確認
