# defamation-monitoring / 誹謗中傷対応・名誉毀損監視

## 概要
オンライン上の誹謗中傷・名誉毀損書き込みを監視し、証拠保全・通報・プラットフォーム別対応手順を管理するツール。Perplexity・Grok APIを活用した検索・分析機能を備える。

## 技術構成
| 項目 | 詳細 |
|------|------|
| Platform | Google Apps Script |
| Language | JavaScript |
| AI APIs | Perplexity API, Grok API |
| 通知 | Email (Gmail) |
| トリガー | 定期実行トリガー |

## ファイル構成
- `名誉毀損書き込み.js` - メインスクリプト（監視、証拠保全、通報処理、プラットフォーム別対応）
- `誹謗中傷対応の流れ.ini` - 誹謗中傷対応フロー・手順書
- `push_to_github.sh` - GitHub連携用スクリプト

## セットアップ
1. Google Apps Scriptプロジェクトを作成
2. `名誉毀損書き込み.js` の内容をスクリプトエディタに貼り付け
3. スクリプトプロパティにAPIキーを設定:
   - `PERPLEXITY_API_KEY`
   - `GROK_API_KEY`
4. 監視対象のキーワード・URLを設定
5. 通知先メールアドレスを設定
6. 定期実行トリガーを設定

## 使い方
- **監視**: トリガーにより定期的にオンライン上の書き込みを自動監視
- **証拠保全**: 名誉毀損に該当する書き込みを検出した場合、自動で証拠を保全
- **通報**: プラットフォーム別の通報手順に従い対応
- **対応フロー**: `誹謗中傷対応の流れ.ini` を参照して法的対応を検討
