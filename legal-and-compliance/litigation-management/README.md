# litigation-management / 訴訟管理システム

## 概要
訴訟案件の一元管理を行うGoogle Apps Scriptツール。案件情報の管理、週次レポートの自動生成、期日チェック、リマインダー通知をスプレッドシート・Slack・Gmail経由で実行する。

## 技術構成
| 項目 | 詳細 |
|------|------|
| Platform | Google Apps Script |
| Language | JavaScript |
| データ管理 | Google Sheets |
| 通知 | Slack Webhook, Gmail |
| トリガー | 時限トリガー（週次レポート、期日チェック） |

## ファイル構成
- `訴訟管理.js` - メインスクリプト（案件管理、週次レポート生成、期日チェック、リマインダー通知）

## セットアップ
1. Google Apps Scriptプロジェクトを作成
2. `訴訟管理.js` の内容をスクリプトエディタに貼り付け
3. 訴訟管理用のスプレッドシートを作成し、スクリプトにシートIDを設定
4. Slack Webhook URLをスクリプトプロパティに登録
5. 時限トリガーを設定（週次レポート・期日チェック用）

## 使い方
- **案件管理**: スプレッドシートに訴訟案件情報を入力・更新
- **週次レポート**: トリガーにより自動で週次サマリーが生成・通知される
- **期日チェック**: 設定した期日が近づくとSlack/Gmailでリマインダーが届く
